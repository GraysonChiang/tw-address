## 台灣地址英譯 Taiwan Address Convert To English


### Installation

```sh
composer require graysonchiang/tw-address
```

### Usage
```php
<?php
require __DIR__ . '/vendor/autoload.php';

use \App\Translator;

$translator = new Translator();

echo $translator->get('台北市信義區忠孝東路一段33號44樓');

```
